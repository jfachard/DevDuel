*   Node.js (v16+)


### 2. Backend Setup
```bash
cd server
npm install
npm start
# Server runs on http://localhost:3001
```

### 3. Frontend Setup
Open a new terminal:
```bash
cd client
npm install
npm run dev
# Client runs on http://localhost:5173
```
